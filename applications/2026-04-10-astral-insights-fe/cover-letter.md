# Cover Letter — Astral Insights, Frontend Developer

The frontend for Voices That Remain is a React 19 application that lets users browse, search, and interact with archival records processed by a 4-stage AI pipeline. The interface is live at voicesthatremain.com. Building a frontend powered by generative AI taught me something that most frontend developers have not encountered yet: LLM outputs are unpredictable in shape and confidence, and the UI needs to account for that. I design interfaces that surface AI-generated content with appropriate context, validation states, and human review hooks rather than treating model outputs as ground truth.

The full project is 58,700 lines of TypeScript across 547 commits, with the AI pipeline using GPT-5.4 structured outputs, tool calling, and vision. I built both the frontend and the backend, which means I understand the entire data path from API call to rendered component. For a consultancy building generative AI solutions, a frontend developer who can reason about the AI layer and not just consume it saves a lot of back-and-forth between teams.

Before I built any of this with AI agents, I wrote 14,000 lines of Python by hand for SonicGen. That deliberate hand-coding phase through 2025 means I can debug generated code, not just ship it. I also reverse-engineered 92 API endpoints for Staffclaw without any documentation, which is the kind of figuring-things-out skill that consulting work demands.

I am based in Raleigh and interested in bringing my frontend-plus-AI experience to Astral Insights. I would like to talk about the types of generative AI solutions you are building for clients.
